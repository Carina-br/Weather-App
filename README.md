# Weather-App
Um aplicativo de clima usando HTML, CSS E JavaScript. Pegamos a informação de temperatura usando a API de OpenWeatherMap e mostra informações de temperatura, condição climática, humidade e velocidade do vento de acordo com a cidade escolhida.
